About CityWeather

CityWeather is a full-stack weather application created as a solo project for INST377 at the University of Maryland.

Project Goal

The goal of this project is to allow users to search for live weather information from cities around the world, compare weather conditions visually, and interact with map-based city locations.

Technologies Used

  • Node.js
  • Express.js
  • Supabase Database
  • OpenWeatherMap API
  • Chart.js
  • Leaflet.js
  • HTML, CSS, JavaScript

Project Features

  • Live weather search
  • Save favorite cities
  • Weather comparison chart
  • Interactive map integration
  • REST API backend
